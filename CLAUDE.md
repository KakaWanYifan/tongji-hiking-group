# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Overview

This is a content repository for the **同济徒步群 (Tongji Hiking Group)**. It stores hiking trip records as Markdown files with accompanying photos.

## Structure

- `txt.txt` — index file listing content categories (出发召集 / 归来记录)
- Each trip is a folder named `YYMMDD-地点` (e.g., `260124-苏州东山莫厘峰`)
  - Contains a Markdown write-up (`YYMMDD-地点.md`) and an `img/` subfolder with photos

## Content Conventions

- Trip records are written in Chinese
- Image paths in Markdown use backslash separators (Windows-style, e.g., `img\莫厘峰-01.jpg`)
- Photos are numbered sequentially (e.g., `莫厘峰-01.jpg`, `莫厘峰-02.jpg`)
- Commit messages are in Chinese
